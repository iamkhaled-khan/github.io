# github.io
For anyone interested in building their own personal portfolio, here’s a quick step-by-step guide to creating your own. 

Steps: 

1️⃣ Download a free HTML template (I used one from HTML5 UP).
2️⃣ Edit the content using a code editor (like VS Code).
3️⃣ Create a GitHub account.
4️⃣ Create a repository named: yourusername.github.io
5️⃣ Upload your website files (make sure index.html is in the root folder).
6️⃣ Enable GitHub Pages in repository settings.
7️⃣ (Optional) Connect a free-form service like Formspree to make the contact section functional.

That’s it, your website will be live at: yourusername.github.io
Happy to help anyone who wants to build theirs!
